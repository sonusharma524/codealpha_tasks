Project: CGPA Calculator in C++
---------------------------------------------


Description:
 This is a simple C++ program that calculates the CGPA (Cumulative Grade Point Average) of a student based on the number of courses taken, grade points received, and              corresponding credit hours.

How It Works:
1. User inputs the number of courses.


2. For each course, the user inputs:
      - Course name
      - Grade point (e.g., A=10, B=8, etc.)
      - Credit hours


3. The program calculates:
       - Total grade points
       - Total credit hours
       - Final CGPA = Total Grade Points / Total Credits

Usage:
Compiler with : Embarcadero  Dev-C++  cgpa_calculator.cpp -o cgpa


Run with:  ./cgpa
